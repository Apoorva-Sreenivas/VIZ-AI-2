Welcome

In today's data-driven world, the ability to interpret and visualize data effectively is crucial for decision-making across various domains. From healthcare and finance to social sciences and technology, the demand for intuitive and insightful data visualizations is ever-increasing. However, creating these visualizations often requires specialized skills and knowledge in data analysis and visualization techniques, which can be a barrier for many users. Leveraging advancements in natural language processing (NLP) and machine learning, the proposed system will allow users to simply describe the data they want to visualize and receive corresponding graphical representations.

About Us

We are a team of passionate Computer Science Engineering students from Bangalore Institute of Technology, driven to bridge the gap between data analysis and visualization. Our mission is to empower users to effortlessly explore and understand complex data insights through intuitive visualizations.

Help

Getting Started

Welcome to our data visualization system! This system allows you to generate visualizations from natural language queries and CSV datasets.

Supported File Format

Our system currently supports CSV (Comma Separated Values) files.

Querying the System

To query the system, first upload a CSV dataset. For example:

  • "Visualize sales by region"
  • "Show customer demographics"
  • "Plot temperature vs. humidity"

Visualization Options

Our system offers various visualization options, including:

  • Bar charts
  • Line charts
  • Pie charts
  • Scatter plots
  • Histograms
  • bubble charts
  • box plots

Uploading CSV Files

  1. Click the "Upload File" button.
  2. Select your CSV file.
  3. Wait for the file to upload.

Troubleshooting

Common issues:

  • Check file format (only CSV supported).
  • Ensure data is correctly formatted.
  • If an accurate visualization is not generated, then explicitly specify the chart name and fields from the list displayed beside the chat window
  • Contact support for further assistance.
